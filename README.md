# Agente de consulta de DataFrame em Python 
 Este repositório contém um agente de consulta de DataFrame que permite aos usuários interagir com DataFrames do Pandas usando linguagem natural. O agente utiliza o modelo GPT-4 da OpenAI para interpretar consultas e gerar código Python correspondente.

 ## Funcionalidades
 - Carregamento de DataFrames a partir de arquivos CSV.
 - Interpretação de consultas em linguagem natural.
 - Geração automática de código Python para manipulação de DataFrames.
 - Execução segura do código gerado.
 - Retorno dos resultados das consultas ao usuário.

 ## Requisitos
 - Python 3.8 ou superior
 - Bibliotecas: pandas e openai
